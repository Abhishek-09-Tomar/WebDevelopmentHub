HTML Links - Hyperlinks

Introduction

HTML links are hyperlinks. You can click on a link and jump to another document or website.

Examples of Links

Visit Google Visit Facebook Visit Instagram Visit Abhishek's Portfolio

Target Attribute Usage

We have 4 types of target attributes that decide where the link will open:

  • _self : Opens the link in the same page (default).
  • _blank : Opens the link in a new tab.
  • _parent : Opens the link in the parent frame (useful in nested frames).
  • _top : Opens the link in the full body of the window, removing any frames.

Relative URL vs Absolute URL

Relative URL: Specifies the location of a resource relative to the current page. It does not include the domain name.

  • Relative Example 1: about.html
  • Relative Example 2: assets/model.jpg

Absolute URL: Specifies the complete address of a resource, including protocol (http/https), domain, and full path.

  • Absolute Example 1: https://www.google.com
  • Absolute Example 2: https://www.wikipedia.org/images/logo.png
