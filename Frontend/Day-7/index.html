HTML Styles - CSS

CSS stands for Cascading Style Sheets.

What is CSS?

CSS is used to format the layout of a webpage.
With CSS, you can control the color, font, the size of text, the spacing between elements, how elements are positioned and laid out, what background images or background colors are to be used, different displays for different devices and screen sizes, and much more!

Tip: The word cascading means that a style applied to a parent element will also apply to all children elements within the parent (unless overridden).

Ways of using CSS in HTML

  • Inline - by using the style attribute inside HTML elements.
  • Internal - by using a <style> element in the <head> section.
  • External - by using a <link> element to link to an external CSS file.

3. External CSS

An external style sheet is used to define the style for many HTML pages at once.
You create a .css file and link it in the head section using: 

<link rel="stylesheet" href="styles.css">

Example "styles.css" file:

            body {
            background-color: powderblue;
            }
            h1 {
            color: blue;
            }
            p {
            color: red;
            }
