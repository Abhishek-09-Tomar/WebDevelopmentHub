Let's Talk About Headings and Why We Use Them?

  • Heading 1 (h1): The most important heading. It usually contains your website’s main topic or name and helps with SEO.
  • Heading 2 (h2): Used for main sections of a page. It organizes content under the h1 heading.
  • Heading 3 (h3): Used for sub-sections under h2 headings. It helps break content into smaller parts.
  • Heading 4 (h4): Used for smaller sub-sections under h3. Useful for detailed topics.
  • Heading 5 (h5): Used for minor headings, usually under h4. Helps organize fine details.
  • Heading 6 (h6): The least important heading. Used for very small sub-sections or notes.

HTML Horizontal Rules

The (hr) tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
The hr element is used to separate content (or define a change) in an HTML page:

Example :

This is part one.

This is part two.

This is part three.

NOTE : As you can see above we are dividing the three parts with the help of hr tag. That is why we are using this hr tag.
  • The hr tag is an empty tag, which means that it has no end tag.
