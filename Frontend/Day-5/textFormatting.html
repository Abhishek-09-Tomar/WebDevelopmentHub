HTML Formatting Elements

  • Bold text → makes the text appear thicker and darker.
  • Important text → looks bold, but also tells the browser the text is important.
  • Italic text → slants the text to the right.
  • Emphasized text → looks italic, but also gives meaning that the text should be stressed.
  • Marked text → highlights the text, usually with a yellow background.
  • Smaller text → reduces the size of the text compared to normal text.
  • Deleted text → shows text with a line through it (strikethrough).
  • Inserted text → shows text with an underline, indicating it was added.
  • H2O → Subscript text appears smaller and slightly below the normal line.
  • 25 → Superscript text appears smaller and slightly above the normal line.

Difference Between Bold and Strong

This text is bold.

This text is Strong.

Note : Bold is used to simply make text appear thicker and darker for visual styling.

Strong is used to indicate that the text is of high importance or emphasis. While it also appears bold by default, it carries semantic meaning for browsers, screen readers, and search engines.
